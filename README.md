# Notes App (Android)

A simple and user-friendly Notes Management Android application developed using Kotlin.  
This app allows users to create, edit, and delete notes with persistent local storage.

## Features

- Add new notes
- Edit existing notes
- Delete notes
- View notes using RecyclerView
- Persistent data storage
- Simple and clean UI

## Technologies Used

- Kotlin
- Android SDK
- RecyclerView
- ViewBinding
